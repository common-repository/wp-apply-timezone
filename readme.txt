== Description ==

Adds the option to apply the current blog's timezone retroactively to existing posts.

Useful if you had the wrong timezone set for a period. Creates an "Apply Timezone" option to the Edit Post Bulk Action menu.

This plugin has had limited testing. Make a full BACKUP before installing. The plugin has worked fine for me, but it comes with no guarantee whatsoever. Please be careful. You can use the <a href="http://wordpress.org/extend/plugins/wp-db-backup/">WP-DB-Backup</a>.

== Installation ==

Upload, activate, as usual.

Backup your database before starting.
